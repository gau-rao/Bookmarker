# Spring-with-Kubernetes

> **NOTE:**
> The code in the `main` branch is not updated to use the latest versions of the dependencies.

## How to run?

```shell

$ ./run.sh start
$ ./run.sh stop

$ ./run.sh start_infra
$ ./run.sh stop_infra
```

* To start only dependent services

```shell
$ ./run.sh start_infra
$ ./run.sh stop_infra
```
